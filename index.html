What is HTML?

HTML- HyperText Markup Language, sounds difficult, so let’s try to explain what it means easier.

In HTML we have next technical stuff:

  1. Element
  2. Tag
    1. tag name
    2. start tag
    3. end tag
    4. self closing tag
  3. Attribute
    1. Attribute name
    2. Attribute value
  4. Content

Small explaining what these elements means:

The <!DOCTYPE html> declaration defines that this document is an HTML5 document

The <html> element is the root element of an HTML page

The <title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab)

The <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

The <div> element. You use the div tag in HTML to divide sections of an HTML document.

After opening your index.html in your browser you should to see same result:

  1. Your content in title element
  2. path to your file which you open
  3. Your div with content

Tags can be self closing or not, for example you can write tag div in two ways

  1. <div></div>
  2. <div/>

We have a list of tags which is self closing: area, base, br, col, embed, hr, img, input, link, meta, param, source, track, wbr, command, keygen, menuitem
